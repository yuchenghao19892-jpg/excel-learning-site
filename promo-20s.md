## Hyperframes-style Prompt

Create a 20-second 16:9 product promo for a Chinese website called "Excel 学习地图". The product teaches Excel through practical daily tasks. Start with messy spreadsheet data transforming into clean tables, then reveal the website hero with the headline "把 Excel 学成每天都用得上的能力". Show four learning stages: 基础操作, 公式函数, 数据分析, 自动化办公. Show a function search UI finding XLOOKUP and SUMIFS. Show practice cards filtering to 实战 and a sales dashboard example. End on a clean call-to-action: "每天 30 分钟，练出办公硬实力". Use crisp modern UI motion, green and teal accents, realistic spreadsheet dashboards, upbeat confident pacing, no cartoon style, no stock-photo people.
