Learning Path

四段式学习路径

每一阶段都对应一个可交付成果，学完不是“懂了”，而是能做出来。

01

表格基本功

掌握数据录入、格式、筛选、排序、冻结窗格、条件格式和打印设置。

  • 整理一张混乱名单
  • 制作可打印的课程表
  • 用条件格式发现异常值
02

公式与函数

学会用函数解决查找、统计、文本处理、日期计算和跨表引用。

  • SUMIFS / COUNTIFS / IF
  • XLOOKUP / INDEX MATCH
  • TEXTSPLIT / FILTER / UNIQUE
03

数据分析报表

用数据透视表、图表和切片器做能回答问题的销售或运营看板。

  • 搭建月度销售仪表盘
  • 比较区域、产品和渠道
  • 让报表可筛选、可复用
04

效率与自动化

学习 Power Query、模板化工作簿和基础宏思路，把重复劳动变成流程。

  • 批量合并多个文件
  • 清洗日期、空值和重复项
  • 生成每周固定报表

28-Day Plan

四周学习安排

不需要一天学很多，关键是每天完成一个具体动作。

第 1 周

整理数据

格式、筛选、排序、条件格式、表格对象。

第 2 周

写对公式

IF、SUMIFS、XLOOKUP、文本函数、日期函数。

第 3 周

看懂数据

透视表、图表、切片器、动态汇总。

第 4 周

做成模板

Power Query、保护工作表、复用报表流程。
